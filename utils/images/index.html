    resize_image

    def resize_image(img, max_size, upscale)

    Resizes image while keeping aspect ratio.

    The smaller dimension will be resized to max_size, i.e. a 400x500px image with max_size=300 will be resized to 300x375px.

    Args

    • img: Pillow image object.

    • max_size: Maximum width or height of resized image.

    • upscale: If True will upscale small images to max_size.
